LaTeX数学公式

Typora插入公式

Typora可插入行内公式和公式块。

  • 行内公式：可与其他文字放在一行，放在之间，可直接输入$$$$，并输入LaTeX公式
  • 公式块：单独占据一行或多行，放在$$$ $$$之间，输入$$$$然后回车，即可输入

说明

Typora使用LaTeX数学公式，常使用斜线 \+字母 来表示符号或运算，如\div表示除运算，\theta表示θ符号。

并使用{}来包裹需要当做整体的公式、或字母组合。

其他

多行等式

将多行等式对齐，使用\begin{aligned} \end{aligned}，使用\\换行，使用&标明对齐等式。

混合字体

  • 斜体字符 x y z：$ x y z$
  • 非斜体字符 \text{x y z}：$\text{x y z}$

常使用\text来做注释，注释文本不会被识别为公式，不用斜体展示。

空格

  • \quad：空格，约为两个m的宽度, 宽度为：|$\quad$|

  • \qquad，两个\quad空格宽度

  • \：大空格，1/3宽度

上下标记

  • \overline：在表达式上方画一条水平线，\overline{x + y}：$\overline{x + y}$

  • \underline：在表达式下方画一条水平线，\underline{x + y}：$\underline{x + y}$

  • \overbrace：在表达式上方给出一个水平的大括号，\overbrace{1 + 2 + \cdots + n}^{n个}：$\overbrace{1 + 2 + \cdots +n}^{n个}$

  • \underbrace：在表达式下方给出一个水平的大括号，\underbrace{1 + 2 + \cdots + n}_{n个}：$\underbrace{1 + 2 + \cdots +n}_{n个}$
