2-Redis基本操作命令

首先运行服务器端Redis：

查看是否连接到服务器：ping

在客户端使用ping命令，可以查看客户端是否成功连接到服务端。如果成功连接到，会返回一个PONG

切换库命令：select db

Redis默认创建16个库(该配置可以在配置文件中修改)，redis默认使用第一个库。

使用 select db来切换其他库，db代表数据库编号，从0到15。

查看当前库中key的数目：dbsize

dbsize可以返回当前库中，key的数目。

删除当前库数据：flushdb

退出：exit或quit

注意，这个退出，只是指客户端退出，但是服务器端依然在运行。
